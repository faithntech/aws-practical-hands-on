About This Project

This project showcases a secure and automated static website deployment on AWS. The site is stored in a private Amazon S3 bucket and distributed via CloudFront using origin access control to prevent direct public access. Deployment is automated through AWS CodePipeline integrated with GitHub without a build stage.

Architecture Highlights

Private S3 Bucket
CloudFront CDN
Origin Access Control
CodePipeline + GitHub
No Build Stage (Static Site)
