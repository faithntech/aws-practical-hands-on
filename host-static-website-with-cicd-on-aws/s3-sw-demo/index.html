About This Project

This project showcases a secure and automated static website deployment on AWS. The site is stored in a private S3 bucket and distributed via CloudFront using origin access control to prevent direct public access. Deployment is fully automated through AWS CodePipeline integrated with GitHub. Since the application is purely static, the pipeline is optimized without a build stage, enabling faster and more efficient releases.

Architecture Highlights

Private S3 Bucket (No Public Access)
CloudFront CDN (Secure Delivery)
Origin Access Control Enabled
CodePipeline (GitHub Integrated)
No Build Stage (Static Optimization)
